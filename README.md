This Software is a shared platform for sharing student provided learning materials such as: Notes, Modules, and other related items with other students or academically inclined individuals.

It currently features:

- User Accounts
- Flexible uploads (PDF, DOCX, PNG, JPEG) [Will add more in the future]
- Browsable and user friendly pages
- A forum based on student opinions and other topics related to the academe
- A way to download and view downloaded files

Future updates will add/refine more prominent features to aid the app and its users.
